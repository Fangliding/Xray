# QUIC 传输

恢复自被删除的 QUIC 传输模块，下面的文档也是复制来的。删掉了没用的 Header 伪装和加密，如果你要那些东西可能需要的是主线目前在推的 UDP Mask 重复的东西确实没意义所以我删了，然后你会发现一件很离谱的事情，它们被删完了（摊手）。也许简单就是最好的。其他伪装选项我后续找时间再考虑吧，这个模块目前的主要作用理论上是探索维护分支一个传输的可行性。

使用 hy2 同款的 BBR 拥塞控制。（饺子醋）

## QuicObject

`QuicObject` 对应传输配置的 `quicSettings` 项。

::: danger
对接的两端的配置必须完全一致，否则连接失败。
QUIC 强制要求开启 TLS，在传输配置中没有开启 TLS 时，Xray 会自行签发一个证书进行 TLS 通讯。
